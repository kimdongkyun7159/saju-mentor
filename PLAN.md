# 사주멘토 (SajuMentor) — 사주도 풀고, 내 마음도 풀고

## 프로젝트 개요
- **이름**: 사주멘토
- **콘셉트**: 나를 기억하는 AI 사주 상담사
- **기술 스택**: Python 3.13 + FastAPI + SQLite + Jinja2 + Gemini API
- **포트**: 5031
- **API**: Google Gemini 2.0 Flash (메인) + 규칙 기반 해석 (폴백)

## 아키텍처
- 로그인: 닉네임 + 생년월일시 = 고유 식별자
- 대화: Gemini API에 사주 데이터를 시스템 프롬프트로 전달 → 질문에 맞는 답변 생성
- 폴백: Gemini 실패 시 규칙 기반 해석으로 자동 전환
- 명리학: 억부용신론 + 조후보정 (한국 주류)
- 야자시/조자시 지원
